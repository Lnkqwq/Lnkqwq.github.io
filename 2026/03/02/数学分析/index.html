区间套定理

区间套

设闭区间列 ∣[an,bn]∣|[a_n, b_n]|∣[an​,bn​]∣ 具有如下性质：

- (i) \([a_n, b_n] \supseteq [a_{n+1}, b_{n+1}], \quad n = 1, 2, \cdots\)

- (ii) \(\lim_{n \to \infty} (b_n - a_n) = 0\)

则称 ∣[an,bn]∣|[a_n, b_n]|∣[an​,bn​]∣ 为闭区间套，简称区间套。

性质 (i) 表明区间列是前一个套着后一个，即：

a1≤a2≤⋯≤an≤⋯≤bn≤⋯≤b2≤b1a_1 \leq a_2 \leq \cdots \leq a_n \leq \cdots \leq b_n \leq \cdots \leq b_2 \leq b_1 a1​≤a2​≤⋯≤an​≤⋯≤bn​≤⋯≤b2​≤b1​

区间套定理

若 ∣[an,bn]∣|[a_n, b_n]|∣[an​,bn​]∣ 是一个区间套，则在实数系中存在唯一的一点 ξ\xiξ，使得：

ξ∈[an,bn],n=1,2,⋯\xi \in [a_n, b_n], \quad n = 1, 2, \cdots ξ∈[an​,bn​],n=1,2,⋯

即：

an≤ξ≤bn,n=1,2,⋯a_n \leq \xi \leq b_n, \quad n = 1, 2, \cdots an​≤ξ≤bn​,n=1,2,⋯

证明

  1. 由区间套性质 (i) 知 {an}\{a_n\}{an​}是递增有界数列，由单调有界定理，存在极限 ξ\xiξ，且：

    an≤ξ,n=1,2,⋯a_n \leq \xi, \quad n = 1, 2, \cdots an​≤ξ,n=1,2,⋯

  2. 同理，{bn}\{b_n\}{bn​}也是递减有界数列，由区间套条件 (i) 和 (ii) 得：

    lim⁡n→∞bn=lim⁡n→∞an=ξ\lim_{n \to \infty} b_n = \lim_{n \to \infty} a_n = \xi n→∞lim​bn​=n→∞lim​an​=ξ

    且：

    bn≥ξ,n=1,2,⋯b_n \geq \xi, \quad n = 1, 2, \cdots bn​≥ξ,n=1,2,⋯

  3. 联合得：

    an≤ξ≤bn,n=1,2,⋯a_n \leq \xi \leq b_n, \quad n = 1, 2, \cdots an​≤ξ≤bn​,n=1,2,⋯

  4. 唯一性：若存在另一个 ξ′\xi'ξ′ 也满足上式，则：

    ∣ξ−ξ′∣≤bn−an,∀n|\xi - \xi'| \leq b_n - a_n, \quad \forall n ∣ξ−ξ′∣≤bn​−an​,∀n

    由区间套条件 (ii) 得：

    ∣ξ−ξ′∣≤lim⁡n→∞(bn−an)=0⇒ξ′=ξ|\xi - \xi'| \leq \lim_{n \to \infty} (b_n - a_n) = 0 \Rightarrow \xi' = \xi ∣ξ−ξ′∣≤n→∞lim​(bn​−an​)=0⇒ξ′=ξ

推论

若 ξ∈[an,bn]\xi \in [a_n, b_n]ξ∈[an​,bn​] 是区间套所确定的点，则对任意 ε>0\varepsilon > 0ε>0，存在 N>0N > 0N>0，使得当 n>Nn > Nn>N 时，有：

聚点定理

聚点

设 SSS 为数轴上的点集，ξ\xiξ 为定点（它可以属于 SSS，也可以不属于 SSS）。若 ξ\xiξ 的任何邻域都含有 SSS 中无穷多个点，则称 ξ\xiξ 为点集 SSS 的一个聚点。

例如：

  • 点集 S={(−1)n+1n}S = \left\{ (-1)^n + \frac{1}{n} \right\}S={(−1)n+n1​} 有两个聚点 ξ1=−1\xi_1 = -1ξ1​=−1 和 ξ2=1\xi_2 = 1ξ2​=1
  • 点集 S={1n}S = \left\{ \frac{1}{n} \right\}S={n1​} 只有一个聚点 ξ=0\xi = 0ξ=0
  • 若 SSS 为开区间 (a,b)(a, b)(a,b)，则 (a,b)(a, b)(a,b) 上每一点以及端点 a,ba, ba,b 都是 SSS 的聚点
  • 正整数集 N+\mathbb{N}_+N+​ 没有聚点，任何有限数集也没有聚点

聚点定理

定理 7.2（聚点定理） 实轴上的任一有界无限点集 SSS 至少有一个聚点。

（注：聚点定理也称为波尔查诺-魏尔斯特拉斯定理。）

证明思路

设 SSS 是实轴上的有界无限点集，则存在 M>0M > 0M>0，使得 S⊂[−M,M]S \subset [-M, M]S⊂[−M,M]。

采用区间套法证明：

  1. 将区间 [−M,M][-M, M][−M,M] 等分为两个闭区间 [−M,0][-M, 0][−M,0] 和 [0,M][0, M][0,M]，则至少有一个区间含有 SSS 的无限多个点，记该区间为 [a1,b1][a_1, b_1][a1​,b1​]

  2. 将 [a1,b1][a_1, b_1][a1​,b1​] 等分为两个闭区间，同样取含有 SSS 的无限多个点的子区间，记为 [a2,b2][a_2, b_2][a2​,b2​]

  3. 如此无限进行下去，得到一列闭区间 {[an,bn]}\{[a_n, b_n]\}{[an​,bn​]}，满足：

    −(i) [an,bn]⊃[an+1,bn+1],n=1,2,⋯- (i) \ [a_n, b_n] \supset [a_{n+1}, b_{n+1}], \quad n = 1, 2, \cdots −(i) [an​,bn​]⊃[an+1​,bn+1​],n=1,2,⋯

    −(ii) lim⁡n→∞(bn−an)=lim⁡n→∞2M2n−1=0- (ii) \ \lim_{n \to \infty} (b_n - a_n) = \lim_{n \to \infty} \frac{2M}{2^{n-1}} = 0 −(ii) n→∞lim​(bn​−an​)=n→∞lim​2n−12M​=0

    −(iii) 每个[an,bn]都含有S的无限多个点- (iii) \ 每个 [a_n, b_n] 都含有 S 的无限多个点 −(iii) 每个[an​,bn​]都含有S的无限多个点

  4. 由区间套定理，存在唯一的 ξ∈[an,bn], n=1,2,⋯\xi \in [a_n, b_n], \ n = 1, 2, \cdotsξ∈[an​,bn​], n=1,2,⋯

    对任意 ε>0\varepsilon > 0ε>0，由区间套推论，存在 NNN 使得 [aN,bN]⊂U(ξ;ε)[a_N, b_N] \subset U(\xi; \varepsilon)[aN​,bN​]⊂U(ξ;ε)，而 [aN,bN][a_N, b_N][aN​,bN​] 含有 SSS 的无限多个点，故 U(ξ;ε)U(\xi; \varepsilon)U(ξ;ε) 含有 SSS 的无限多个点。因此 ξ\xiξ 是 SSS 的聚点。

推论（波尔查诺-魏尔斯特拉斯定理）

有界数列 {xn}\{x_n\}{xn​} 必有收敛子列。

证明：若 {xn}\{x_n\}{xn​} 只有有限个不同的项，则必存在无穷多项相等，这些相等的项构成收敛子列；若 {xn}\{x_n\}{xn​} 有无限个不同的项，则这些不同项构成有界无限点集，由聚点定理，存在聚点 ξ\xiξ，则存在子列收敛于 ξ\xiξ。

有限覆盖定理

开覆盖

设 SSS 为数轴上的点集，H\mathcal{H}H 为开区间构成的集合（即每个元素都是开区间）。若 SSS 中任何一点都含在 H\mathcal{H}H 中至少一个开区间内，则称 H\mathcal{H}H 是 SSS 的一个开覆盖。

若 H\mathcal{H}H 中只有有限个开区间，则称 H\mathcal{H}H 为有限开覆盖。

有限覆盖定理

定理 7.3（有限覆盖定理） 若 H\mathcal{H}H 是闭区间 [a,b][a, b][a,b] 的一个开覆盖，则从 H\mathcal{H}H 中必可选出有限个开区间，构成 [a,b][a, b][a,b] 的一个有限开覆盖。

（注：有限覆盖定理也称为海涅-波莱尔定理。）

证明（用区间套法反证）

假设结论不成立，即 [a,b][a, b][a,b] 不能被 H\mathcal{H}H 中有限个开区间覆盖。

  1. 记 [a1,b1]=[a,b][a_1, b_1] = [a, b][a1​,b1​]=[a,b]，将其等分为两个闭区间 [a1,a1+b12][a_1, \frac{a_1+b_1}{2}][a1​,2a1​+b1​​] 和 [a1+b12,b1][\frac{a_1+b_1}{2}, b_1][2a1​+b1​​,b1​]，则至少有一个子区间不能被 H\mathcal{H}H 中有限个开区间覆盖，记该区间为 [a2,b2][a_2, b_2][a2​,b2​]

  2. 将 [a2,b2][a_2, b_2][a2​,b2​] 等分为两个闭区间，同样取不能被有限覆盖的子区间，记为 [a3,b3][a_3, b_3][a3​,b3​]

  3. 如此无限进行下去，得到一列闭区间 {[an,bn]}\{[a_n, b_n]\}{[an​,bn​]}，满足：

    −(i) [an,bn]⊃[an+1,bn+1],n=1,2,⋯- (i) \ [a_n, b_n] \supset [a_{n+1}, b_{n+1}], \quad n = 1, 2, \cdots −(i) [an​,bn​]⊃[an+1​,bn+1​],n=1,2,⋯

    −(ii) lim⁡n→∞(bn−an)=lim⁡n→∞b−a2n−1=0- (ii) \ \lim_{n \to \infty} (b_n - a_n) = \lim_{n \to \infty} \frac{b-a}{2^{n-1}} = 0 −(ii) n→∞lim​(bn​−an​)=n→∞lim​2n−1b−a​=0

    −(iii) 每个[an,bn]都不能被H中有限个开区间覆盖- (iii) \ 每个 [a_n, b_n] 都不能被 \mathcal{H} 中有限个开区间覆盖 −(iii) 每个[an​,bn​]都不能被H中有限个开区间覆盖

  4. 由区间套定理，存在唯一的 ξ∈[an,bn], n=1,2,⋯\xi \in [a_n, b_n], \ n = 1, 2, \cdotsξ∈[an​,bn​], n=1,2,⋯，且 ξ∈[a,b]\xi \in [a, b]ξ∈[a,b]

  5. 由于 H\mathcal{H}H 是 [a,b][a, b][a,b] 的开覆盖，存在开区间 (α,β)∈H(\alpha, \beta) \in \mathcal{H}(α,β)∈H 使得 ξ∈(α,β)\xi \in (\alpha, \beta)ξ∈(α,β)

  6. 取 ε=min⁡{ξ−α,β−ξ}>0\varepsilon = \min\{\xi - \alpha, \beta - \xi\} > 0ε=min{ξ−α,β−ξ}>0，由区间套推论，存在 NNN 使得 [aN,bN]⊂U(ξ;ε)⊂(α,β)[a_N, b_N] \subset U(\xi; \varepsilon) \subset (\alpha, \beta)[aN​,bN​]⊂U(ξ;ε)⊂(α,β)

  7. 这意味着 [aN,bN][a_N, b_N][aN​,bN​] 被一个开区间 (α,β)∈H(\alpha, \beta) \in \mathcal{H}(α,β)∈H 覆盖，与性质 (iii) 矛盾

应用举例

有限覆盖定理常用来将局部性质推广到整体。例如，若函数 (f) 在闭区间 ([a, b]) 上每点连续，则对每点存在邻域使得函数值变化很小，这些邻域构成 ([a, b]) 的一个开覆盖，由有限覆盖定理可选出有限个邻域，从而证明 (f) 在 ([a, b]) 上一致连续。

实数完备性定理等价性

六大基本定理

实数系的完备性（连续性）可以通过以下六个基本定理来描述，它们彼此等价：

  1. 确界原理：非空有上界的数集必有上确界；非空有下界的数集必有下确界。
  2. 单调有界定理：单调有界数列必有极限。
  3. 柯西收敛准则：数列收敛的充要条件是它是柯西列。
  4. 区间套定理：闭区间套存在唯一的公共点。
  5. 聚点定理（波尔查诺-魏尔斯特拉斯定理）：有界无限点集至少有一个聚点。
  6. 有限覆盖定理（海涅-波莱尔定理）：闭区间的任何开覆盖必存在有限子覆盖。

上极限和下极限

从聚点到上/下极限

回顾：由聚点定理，有界无限点集至少有一个聚点。对于有界数列 {xn}\{x_n\}{xn​}，其所有收敛子列的极限值称为该数列的极限点（聚点）[citation:6]。

定理：有界数列 {xn}\{x_n\}{xn​} 必存在最大极限点和最小极限点 [citation:6]。

上极限与下极限的定义

定义 1（基于聚点）：
设 {xn}\{x_n\}{xn​} 是有界数列，称其所有收敛子列的极限中的最大值为 {xn}\{x_n\}{xn​} 的上极限，最小值为 {xn}\{x_n\}{xn​} 的下极限 [citation:5][citation:6]。记为：

\varlimsup_{n \to \infty} x_n \quad \text{或} \quad \limsup_{n \to \infty} x_n

\varliminf_{n \to \infty} x_n \quad \text{或} \quad \liminf_{n \to \infty} x_n

定义 2（基于确界）：
设 {xn}\{x_n\}{xn​} 是有界数列，令

hn=sup⁡k≥nxk,mn=inf⁡k≥nxkh_n = \sup_{k \geq n} x_k, \quad m_n = \inf_{k \geq n} x_k hn​=k≥nsup​xk​,mn​=k≥ninf​xk​

则 {hn}\{h_n\}{hn​} 单调递减，{mn}\{m_n\}{mn​} 单调递增 [citation:1][citation:5]。定义：

\varlimsup_{n \to \infty} x_n = \lim_{n \to \infty} h_n = \inf_{n \geq 1} \sup_{k \geq n} x_k

\varliminf_{n \to \infty} x_n = \lim_{n \to \infty} m_n = \sup_{n \geq 1} \inf_{k \geq n} x_k

注：若 {xn}\{x_n\}{xn​} 无上界，则记 \varlimsup_{n \to \infty} x_n = +\infty；若 {xn}\{x_n\}{xn​} 无下界，则记 \varliminf_{n \to \infty} x_n = -\infty [citation:1][citation:7]。

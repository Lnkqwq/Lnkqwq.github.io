一

题目：

  1. ∫sec⁡3x dx\displaystyle \int \sec^3 x \, dx∫sec3xdx
  2. ∫x2+a2 dx(a>0)\displaystyle \int \sqrt{x^2 + a^2} \, dx \quad (a > 0)∫x2+a2​dx(a>0)
  3. ∫x2−a2 dx(a>0)\displaystyle \int \sqrt{x^2 - a^2} \, dx \quad (a > 0)∫x2−a2​dx(a>0)

解

(1) ∫sec⁡3x dx\int \sec^3 x \, dx∫sec3xdx

∫sec⁡3x dx=∫sec⁡x⋅sec⁡2x dx\int \sec^3 x \, dx = \int \sec x \cdot \sec^2 x \, dx ∫sec3xdx=∫secx⋅sec2xdx

令u=sec⁡x，dv=sec⁡2x dxu = \sec x，dv = \sec^2 x \, dxu=secx，dv=sec2xdx，则

du=sec⁡xtan⁡x dx，v=tan⁡x。du = \sec x \tan x \, dx，v = \tan x。 du=secxtanxdx，v=tanx。

∫sec⁡3x dx=sec⁡xtan⁡x−∫sec⁡xtan⁡2x dx\int \sec^3 x \, dx = \sec x \tan x - \int \sec x \tan^2 x \, dx ∫sec3xdx=secxtanx−∫secxtan2xdx

tan⁡2x=sec⁡2x−1\tan^2 x = \sec^2 x - 1 tan2x=sec2x−1

∫sec⁡3x dx=sec⁡xtan⁡x−∫sec⁡x(sec⁡2x−1) dx\int \sec^3 x \, dx = \sec x \tan x - \int \sec x (\sec^2 x - 1) \, dx ∫sec3xdx=secxtanx−∫secx(sec2x−1)dx

=sec⁡xtan⁡x−∫sec⁡3x dx+∫sec⁡x dx= \sec x \tan x - \int \sec^3 x \, dx + \int \sec x \, dx =secxtanx−∫sec3xdx+∫secxdx

移项：

2∫sec⁡3x dx=sec⁡xtan⁡x+ln⁡∣sec⁡x+tan⁡x∣+C2 \int \sec^3 x \, dx = \sec x \tan x + \ln|\sec x + \tan x| + C 2∫sec3xdx=secxtanx+ln∣secx+tanx∣+C

∫sec⁡3x dx=12sec⁡xtan⁡x+12ln⁡∣sec⁡x+tan⁡x∣+C\int \sec^3 x \, dx = \frac12 \sec x \tan x + \frac12 \ln|\sec x + \tan x| + C ∫sec3xdx=21​secxtanx+21​ln∣secx+tanx∣+C

(2) ∫x2+a2 dx\int \sqrt{x^2 + a^2} \, dx∫x2+a2​dx

令 x=atan⁡tx = a \tan tx=atant，则 dx=asec⁡2t dt，x2+a2=asec⁡tdx = a \sec^2 t \, dt，\sqrt{x^2 + a^2} = a \sec tdx=asec2tdt，x2+a2​=asect。

∫x2+a2 dx=∫asec⁡t⋅asec⁡2t dt=a2∫sec⁡3t dt\int \sqrt{x^2 + a^2} \, dx = \int a \sec t \cdot a \sec^2 t \, dt = a^2 \int \sec^3 t \, dt ∫x2+a2​dx=∫asect⋅asec2tdt=a2∫sec3tdt

由 (1) 的结果：

∫sec⁡3t dt=12sec⁡ttan⁡t+12ln⁡∣sec⁡t+tan⁡t∣+C\int \sec^3 t \, dt = \frac12 \sec t \tan t + \frac12 \ln|\sec t + \tan t| + C ∫sec3tdt=21​secttant+21​ln∣sect+tant∣+C

代回：

∫x2+a2 dx=a2(12sec⁡ttan⁡t+12ln⁡∣sec⁡t+tan⁡t∣)+C\int \sqrt{x^2 + a^2} \, dx = a^2 \left( \frac12 \sec t \tan t + \frac12 \ln|\sec t + \tan t| \right) + C ∫x2+a2​dx=a2(21​secttant+21​ln∣sect+tant∣)+C

sec⁡t=x2+a2a,tan⁡t=xa\sec t = \frac{\sqrt{x^2 + a^2}}{a}, \quad \tan t = \frac{x}{a} sect=ax2+a2​​,tant=ax​

∫x2+a2 dx=12xx2+a2+a22ln⁡∣x+x2+a2a∣+C\int \sqrt{x^2 + a^2} \, dx = \frac12 x \sqrt{x^2 + a^2} + \frac{a^2}{2} \ln\left| \frac{x + \sqrt{x^2 + a^2}}{a} \right| + C ∫x2+a2​dx=21​xx2+a2​+2a2​ln∣∣∣∣∣∣​ax+x2+a2​​∣∣∣∣∣∣​+C

常数 a22ln⁡a可并入C，得：\frac{a^2}{2} \ln a 可并入 C，得：2a2​lna可并入C，得：

∫x2+a2 dx=12xx2+a2+a22ln⁡∣x+x2+a2∣+C\boxed{\int \sqrt{x^2 + a^2} \, dx = \frac12 x \sqrt{x^2 + a^2} + \frac{a^2}{2} \ln\left| x + \sqrt{x^2 + a^2} \right| + C} ∫x2+a2​dx=21​xx2+a2​+2a2​ln∣∣∣∣​x+x2+a2​∣∣∣∣​+C​

(3) ∫x2−a2 dx\int \sqrt{x^2 - a^2} \, dx∫x2−a2​dx

令 x=asec⁡t，则dx=asec⁡ttan⁡t dt，x2−a2=atan⁡t。x = a \sec t，则 dx = a \sec t \tan t \, dt，\sqrt{x^2 - a^2} = a \tan t。x=asect，则dx=asecttantdt，x2−a2​=atant。

∫x2−a2 dx=∫atan⁡t⋅asec⁡ttan⁡t dt=a2∫sec⁡ttan⁡2t dt\int \sqrt{x^2 - a^2} \, dx = \int a \tan t \cdot a \sec t \tan t \, dt = a^2 \int \sec t \tan^2 t \, dt ∫x2−a2​dx=∫atant⋅asecttantdt=a2∫secttan2tdt

tan⁡2t=sec⁡2t−1\tan^2 t = \sec^2 t - 1 tan2t=sec2t−1

∫x2−a2 dx=a2(∫sec⁡3t dt−∫sec⁡t dt)\int \sqrt{x^2 - a^2} \, dx = a^2 \left( \int \sec^3 t \, dt - \int \sec t \, dt \right) ∫x2−a2​dx=a2(∫sec3tdt−∫sectdt)

由 (1) 和 ∫sec⁡t dt=ln⁡∣sec⁡t+tan⁡t∣：\int \sec t \, dt = \ln|\sec t + \tan t|：∫sectdt=ln∣sect+tant∣：

∫sec⁡3t dt−∫sec⁡t dt=12sec⁡ttan⁡t−12ln⁡∣sec⁡t+tan⁡t∣\int \sec^3 t \, dt - \int \sec t \, dt = \frac12 \sec t \tan t - \frac12 \ln|\sec t + \tan t| ∫sec3tdt−∫sectdt=21​secttant−21​ln∣sect+tant∣

代回：

∫x2−a2 dx=a2(12sec⁡ttan⁡t−12ln⁡∣sec⁡t+tan⁡t∣)+C\int \sqrt{x^2 - a^2} \, dx = a^2 \left( \frac12 \sec t \tan t - \frac12 \ln|\sec t + \tan t| \right) + C ∫x2−a2​dx=a2(21​secttant−21​ln∣sect+tant∣)+C

sec⁡t=xa,tan⁡t=x2−a2a\sec t = \frac{x}{a}, \quad \tan t = \frac{\sqrt{x^2 - a^2}}{a} sect=ax​,tant=ax2−a2​​

∫x2−a2 dx=12xx2−a2−a22ln⁡∣x+x2−a2∣+C\int \sqrt{x^2 - a^2} \, dx = \frac12 x \sqrt{x^2 - a^2} - \frac{a^2}{2} \ln\left| x + \sqrt{x^2 - a^2} \right| + C ∫x2−a2​dx=21​xx2−a2​−2a2​ln∣∣∣∣​x+x2−a2​∣∣∣∣​+C

（注意 ln⁡∣a∣\ln|a|ln∣a∣ 已并入 C）

∫x2−a2 dx=12xx2−a2−a22ln⁡∣x+x2−a2∣+C\boxed{\int \sqrt{x^2 - a^2} \, dx = \frac12 x \sqrt{x^2 - a^2} - \frac{a^2}{2} \ln\left| x + \sqrt{x^2 - a^2} \right| + C} ∫x2−a2​dx=21​xx2−a2​−2a2​ln∣∣∣∣​x+x2−a2​∣∣∣∣​+C​

已知：

I(m,n)=cos⁡m−1xsin⁡n+1xm+n+m−1m+nI(m−2,n)I(m, n) = \frac{\cos^{m-1} x \sin^{n+1} x}{m+n} + \frac{m-1}{m+n} I(m-2, n) I(m,n)=m+ncosm−1xsinn+1x​+m+nm−1​I(m−2,n)

和

J=I(m−2,n)−I(m,n)J = I(m-2, n) - I(m, n) J=I(m−2,n)−I(m,n)

cos⁡m−1xsin⁡n+1x=−(m−1)[I(m−2,n)−I(m,n)]+(n+1)I(m,n)\cos^{m-1} x \sin^{n+1} x = -(m-1)[I(m-2, n) - I(m, n)] + (n+1) I(m, n) cosm−1xsinn+1x=−(m−1)[I(m−2,n)−I(m,n)]+(n+1)I(m,n)

=−(m−1)I(m−2,n)+(m−1)I(m,n)+(n+1)I(m,n)= -(m-1) I(m-2, n) + (m-1) I(m, n) + (n+1) I(m, n) =−(m−1)I(m−2,n)+(m−1)I(m,n)+(n+1)I(m,n)

=−(m−1)I(m−2,n)+(m+n)I(m,n)= -(m-1) I(m-2, n) + (m+n) I(m, n) =−(m−1)I(m−2,n)+(m+n)I(m,n)

所以：

(m+n)I(m,n)=cos⁡m−1xsin⁡n+1x+(m−1)I(m−2,n)(m+n) I(m, n) = \cos^{m-1} x \sin^{n+1} x + (m-1) I(m-2, n) (m+n)I(m,n)=cosm−1xsinn+1x+(m−1)I(m−2,n)

I(m,n)=cos⁡m−1xsin⁡n+1xm+n+m−1m+nI(m−2,n)\boxed{I(m, n) = \frac{\cos^{m-1} x \sin^{n+1} x}{m+n} + \frac{m-1}{m+n} I(m-2, n)} I(m,n)=m+ncosm−1xsinn+1x​+m+nm−1​I(m−2,n)​

设 g(x)=cos⁡m+1xsin⁡n−1x,g(x) = \cos^{m+1} x \sin^{n-1} x,g(x)=cosm+1xsinn−1x,求导：

g′(x)=(m+1)cos⁡mx(−sin⁡x)sin⁡n−1x+cos⁡m+1x(n−1)sin⁡n−2xcos⁡xg'(x) = (m+1)\cos^{m} x (-\sin x) \sin^{n-1} x + \cos^{m+1} x (n-1) \sin^{n-2} x \cos x g′(x)=(m+1)cosmx(−sinx)sinn−1x+cosm+1x(n−1)sinn−2xcosx

两边积分：

cos⁡m+1xsin⁡n−1x=−(m+1)I(m,n)+(n−1)∫cos⁡m+2xsin⁡n−2x dx\cos^{m+1} x \sin^{n-1} x = -(m+1) I(m, n) + (n-1) \int \cos^{m+2} x \sin^{n-2} x \, dx cosm+1xsinn−1x=−(m+1)I(m,n)+(n−1)∫cosm+2xsinn−2xdx

K=∫cos⁡mxsin⁡n−2x dx−∫cos⁡mxsin⁡nx dxK = \int \cos^{m} x \sin^{n-2} x \, dx - \int \cos^{m} x \sin^{n} x \, dx K=∫cosmxsinn−2xdx−∫cosmxsinnxdx

cos⁡m+1xsin⁡n−1x=−(m+1)I(m,n)+(n−1)[I(m,n−2)−I(m,n)]\cos^{m+1} x \sin^{n-1} x = -(m+1) I(m, n) + (n-1)[I(m, n-2) - I(m, n)] cosm+1xsinn−1x=−(m+1)I(m,n)+(n−1)[I(m,n−2)−I(m,n)]

=−(m+1)I(m,n)+(n−1)I(m,n−2)−(n−1)I(m,n)= -(m+1) I(m, n) + (n-1) I(m, n-2) - (n-1) I(m, n) =−(m+1)I(m,n)+(n−1)I(m,n−2)−(n−1)I(m,n)

第二部分：应用

题目：

  1. ∫tan⁡3x dx\displaystyle \int \tan^3 x \, dx∫tan3xdx
  2. ∫tan⁡4x dx\displaystyle \int \tan^4 x \, dx∫tan4xdx
  3. ∫cos⁡2xsin⁡4x dx\displaystyle \int \cos^2 x \sin^4 x \, dx∫cos2xsin4xdx

(1) ∫tan⁡3x dx\int \tan^3 x \, dx∫tan3xdx

tan⁡3x=cos⁡−3xsin⁡3x，此时m=−3,n=3，m+n=0，\tan^3 x = \cos^{-3} x \sin^3 x，此时 m = -3, n = 3，m+n=0，tan3x=cos−3xsin3x，此时m=−3,n=3，m+n=0，递推公式不能用，直接计算：

∫tan⁡3x dx=∫tan⁡xsec⁡2x dx−∫tan⁡x dx\int \tan^3 x \, dx = \int \tan x \sec^2 x \, dx - \int \tan x \, dx ∫tan3xdx=∫tanxsec2xdx−∫tanxdx

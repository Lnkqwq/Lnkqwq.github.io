质点运动学

一、位置矢量

定义：从坐标原点指向质点位置的有向线段，描述质点位置的物理量。

· 直角坐标：r=xi+yj+zk\boldsymbol{r} = x\boldsymbol{i} + y\boldsymbol{j} + z\boldsymbol{k}r=xi+yj+zk
· 大小：∣r∣=x2+y2+z2|\boldsymbol{r}| = \sqrt{x^2 + y^2 + z^2}∣r∣=x2+y2+z2​
· 方向：由方向余弦 cos⁡α=x∣r∣\cos\alpha = \dfrac{x}{|\boldsymbol{r}|}cosα=∣r∣x​，cos⁡β=y∣r∣\cos\beta = \dfrac{y}{|\boldsymbol{r}|}cosβ=∣r∣y​，cos⁡γ=z∣r∣\cos\gamma = \dfrac{z}{|\boldsymbol{r}|}cosγ=∣r∣z​ 确定

二、位移

定义：由初位置指向末位置的有向线段，描述质点位置变化的物理量。

Δr=r(t+Δt)−r(t)\Delta \boldsymbol{r} = \boldsymbol{r}(t + \Delta t) - \boldsymbol{r}(t) Δr=r(t+Δt)−r(t)

注意：位移是矢量，路程是标量。一般 ∣Δr∣≠Δs|\Delta \boldsymbol{r}| \neq \Delta s∣Δr∣=Δs，只有在直线运动且方向不变时才相等。

三、速度

描述质点位置变化快慢的物理量。

  1. 平均速度

v‾=ΔrΔt\overline{\boldsymbol{v}} = \frac{\Delta \boldsymbol{r}}{\Delta t} v=ΔtΔr​

  1. 瞬时速度（速度）

v=lim⁡Δt→0ΔrΔt=drdt\boldsymbol{v} = \lim_{\Delta t \to 0} \frac{\Delta \boldsymbol{r}}{\Delta t} = \frac{d\boldsymbol{r}}{dt} v=Δt→0lim​ΔtΔr​=dtdr​

· 方向：沿轨迹切线，指向运动方向
· 大小（速率）：v=∣v∣=dsdtv = |\boldsymbol{v}| = \dfrac{ds}{dt}v=∣v∣=dtds​

  1. 直角坐标系中的速度

v=dxdti+dydtj+dzdtk=vxi+vyj+vzk\boldsymbol{v} = \frac{dx}{dt}\boldsymbol{i} + \frac{dy}{dt}\boldsymbol{j} + \frac{dz}{dt}\boldsymbol{k} = v_x\boldsymbol{i} + v_y\boldsymbol{j} + v_z\boldsymbol{k} v=dtdx​i+dtdy​j+dtdz​k=vx​i+vy​j+vz​k

  1. 极坐标系中的速度

径向速度：

vr=drdtv_r = \frac{dr}{dt} vr​=dtdr​

横向速度：

vθ=rdθdtv_\theta = r\frac{d\theta}{dt} vθ​=rdtdθ​

速度大小：

v=vr2+vθ2=(drdt)2+(rdθdt)2v = \sqrt{v_r^2 + v_\theta^2} = \sqrt{\left(\frac{dr}{dt}\right)^2 + \left(r\frac{d\theta}{dt}\right)^2} v=vr2​+vθ2​​=(dtdr​)2+(rdtdθ​)2​

四、加速度

描述速度变化快慢的物理量。

  1. 平均加速度

a‾=ΔvΔt\overline{\boldsymbol{a}} = \frac{\Delta \boldsymbol{v}}{\Delta t} a=ΔtΔv​

  1. 瞬时加速度（加速度）

a=lim⁡Δt→0ΔvΔt=dvdt=d2rdt2\boldsymbol{a} = \lim_{\Delta t \to 0} \frac{\Delta \boldsymbol{v}}{\Delta t} = \frac{d\boldsymbol{v}}{dt} = \frac{d^2\boldsymbol{r}}{dt^2} a=Δt→0lim​ΔtΔv​=dtdv​=dt2d2r​

  1. 直角坐标系中的加速度

a=dvxdti+dvydtj+dvzdtk=axi+ayj+azk\boldsymbol{a} = \frac{dv_x}{dt}\boldsymbol{i} + \frac{dv_y}{dt}\boldsymbol{j} + \frac{dv_z}{dt}\boldsymbol{k} = a_x\boldsymbol{i} + a_y\boldsymbol{j} + a_z\boldsymbol{k} a=dtdvx​​i+dtdvy​​j+dtdvz​​k=ax​i+ay​j+az​k

  1. 自然坐标系中的加速度

将加速度分解为切向和法向分量：

a=atτ+ann\boldsymbol{a} = a_t \boldsymbol{\tau} + a_n \boldsymbol{n} a=at​τ+an​n

· 切向加速度：at=dvdta_t = \dfrac{dv}{dt}at​=dtdv​，反映速度大小的变化
· 法向加速度：an=v2ρa_n = \dfrac{v^2}{\rho}an​=ρv2​，ρ\rhoρ 为曲率半径，反映速度方向的变化

加速度大小：

a=at2+an2a = \sqrt{a_t^2 + a_n^2} a=at2​+an2​​

二、圆周运动

  1. 线速度

大小：

v=dsdt=lim⁡Δt→0ΔsΔtv = \frac{ds}{dt} = \lim_{\Delta t \to 0}\frac{\Delta s}{\Delta t} v=dtds​=Δt→0lim​ΔtΔs​

方向：沿圆周切线方向，指向运动方向。

  1. 角速度

大小：

ω=dθdt\omega = \frac{d\theta}{dt} ω=dtdθ​

方向：由右手螺旋定则确定，与转动平面垂直。

  1. 线速度与角速度的关系

大小关系：

v=ωrv = \omega r v=ωr

方向关系：v=ω×r\boldsymbol{v} = \boldsymbol{\omega} \times \boldsymbol{r}v=ω×r

  1. 圆周运动的加速度

切向加速度：

at=dvdt=rdωdt=rαa_t = \frac{dv}{dt} = r\frac{d\omega}{dt} = r\alpha at​=dtdv​=rdtdω​=rα

（α\alphaα 为角加速度）

法向加速度（向心加速度）：

an=v2r=ω2ra_n = \frac{v^2}{r} = \omega^2 r an​=rv2​=ω2r

总加速度：

a=atτ+ann\boldsymbol{a} = a_t\boldsymbol{\tau} + a_n\boldsymbol{n} a=at​τ+an​n

大小：

a=at2+an2a = \sqrt{a_t^2 + a_n^2} a=at2​+an2​​

注意：

· 匀速圆周运动：at=0a_t = 0at​=0，an=v2r≠0a_n = \dfrac{v^2}{r} \neq 0an​=rv2​=0
· 变速圆周运动：at≠0a_t \neq 0at​=0，an≠0a_n \neq 0an​=0
· 总加速度方向不指向圆心，仅在匀速圆周运动时总加速度才指向圆心

三、相对运动

绝对运动：质点相对于静止参考系的运动
相对运动：质点相对于运动参考系的运动
牵连运动：运动参考系相对于静止参考系的运动

位移关系：

r绝=r相+r牵\boldsymbol{r}_{\text{绝}} = \boldsymbol{r}_{\text{相}} + \boldsymbol{r}_{\text{牵}} r绝​=r相​+r牵​

速度关系：

v绝=v相+v牵\boldsymbol{v}_{\text{绝}} = \boldsymbol{v}_{\text{相}} + \boldsymbol{v}_{\text{牵}} v绝​=v相​+v牵​

加速度关系：

a绝=a相+a牵\boldsymbol{a}_{\text{绝}} = \boldsymbol{a}_{\text{相}} + \boldsymbol{a}_{\text{牵}} a绝​=a相​+a牵​

基本假设：

· 长度绝对性：在不同参考系中测量同一物体的长度相同
· 时间绝对性：在不同参考系中测量同一过程的时间间隔相同

四、牛顿运动定律

  1. 牛顿第一定律
    任何物体都保持静止或匀速直线运动状态，直到外力迫使它改变这种状态为止。

  2. 牛顿第二定律

F=ma=dpdt\boldsymbol{F} = m\boldsymbol{a} = \frac{d\boldsymbol{p}}{dt} F=ma=dtdp​

其中 p=mv\boldsymbol{p} = m\boldsymbol{v}p=mv 为动量。

  1. 牛顿第三定律
    两个物体之间的作用力 F\boldsymbol{F}F 和反作用力 F′\boldsymbol{F}'F′，沿同一直线，大小相等，方向相反，分别作用在两个物体上。

F=−F′\boldsymbol{F} = -\boldsymbol{F}' F=−F′

  1. 选对象：确定研究对象
  2. 看运动：分析运动状态
  3. 画受力图：分析物体受力
  4. 建坐标系：选择合适的坐标系
  5. 列方程：根据牛顿第二定律列方程
  6. 解方程：求解未知量
  7. 做结论：给出最终结果
